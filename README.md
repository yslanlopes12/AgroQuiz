# 🌱 AgroQuiz – Aprendendo Agricultura Sustentável

O **AgroQuiz** é um aplicativo educacional desenvolvido com **React Native e Expo** que utiliza gamificação para ensinar conceitos de **agricultura sustentável e preservação ambiental**.

Inspirado em aplicativos de aprendizado como o Duolingo, o projeto apresenta um sistema de perguntas interativas onde o usuário aprende enquanto joga. A cada pergunta respondida, o aplicativo fornece feedback visual indicando se a resposta está correta ou incorreta.

O objetivo do projeto é demonstrar um **protótipo de aprendizado gamificado (MVP)** que pode ser utilizado para educação ambiental e capacitação sobre práticas sustentáveis na agricultura.

---

# 🚀 Funcionalidades

- Tela inicial para iniciar o jogo
- Introdução ao tema de **agricultura sustentável**
- Sistema de **quiz com múltiplas escolhas**
- Feedback visual de **acerto ou erro**
- Sistema de **pontuação**
- Tela final com resultado
- Opção de **reiniciar o jogo**
- Banco de perguntas simples utilizando **JSON**

---

# 🛠 Tecnologias utilizadas

- **React Native**
- **Expo**
- **Expo Router**
- **TypeScript**
- **Animated API**
